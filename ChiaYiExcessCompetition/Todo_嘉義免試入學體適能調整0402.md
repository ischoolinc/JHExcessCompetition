# ❗ 問題說明

依規定：

1. 柔軟度：坐姿體前彎
2. 肌力及肌耐力（取最高）：仰臥捲腹 或 一分鐘屈膝仰臥起坐
3. 瞬發力：立定跳遠
4. 心肺耐力（取最高）：800/1600 公尺跑走 或 漸速耐力折返跑

所以：

- `sit_up` 與 `curl` 屬同一類肌耐力項目
- 兩者任一達標，只能算 1 次
- 兩者都達標，仍只能算 1 次
- 不可變成 `+6`
- 不可讓 `addCount +2`

---

# ✅ 正確規則

## 肌耐力類（sit_up / curl）

- 任一達標 → `score += 3`
- 任一達標 → `addCount++`
- 兩者都達標 → 仍只 `score += 3`、`addCount++` 一次
