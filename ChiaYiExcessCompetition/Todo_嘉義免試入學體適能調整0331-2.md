# 🎯 目標

修正 `rptStudentInfo.cs` 中 `CalcFitnessInfoList()` 的體適能計算邏輯：

👉 將
`cardiorespiratory_degree`（心肺耐力）
與
`pacer_degree`（漸速耐力跑）

視為同一類「心肺耐力項目」

---

# ❗ 問題說明

目前程式：

```csharp
if (cardiorespiratory_degreeList.Contains(...))
    score += 3;

if (pacer_degreeList.Contains(...))
    score += 3;
```

👉 當兩者都達標時會變成：

```text
+3 +3 = +6 ❌
```

---

# ✅ 正確規則

```text
cardiorespiratory / pacer 屬同一類

✔ 任一達標 → +3
✔ 兩者都達標 → 仍只 +3
❌ 不可變成 +6
```

---

檔案：

```text
rptStudentInfo.cs
```

方法：

```text
CalcFitnessInfoList()
```

## Step 2：刪除這兩段

👉 **整段刪掉（兩段都要刪）**

---

## Step 3：改成同組判斷

新增以下程式碼：

```csharp
bool cardiorespiratoryPass = false;
foreach (string name in passStringList)
{
    if (cardiorespiratory_degreeList.Contains(name))
    {
        cardiorespiratoryPass = true;
        break;
    }
}

bool pacerPass = false;
foreach (string name in passStringList)
{
    if (pacer_degreeList.Contains(name))
    {
        pacerPass = true;
        break;
    }
}

// 心肺耐力類：只算一次
if (cardiorespiratoryPass || pacerPass)
{
    score += 3;
}
```
