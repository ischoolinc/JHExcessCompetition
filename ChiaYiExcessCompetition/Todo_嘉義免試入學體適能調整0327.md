原本通常會像這樣：

```csharp

    addCount++;
```

---

## 3.2 保留原本 4 項判斷

原本以下 4 項的判斷 **不要刪、不要改原意**：

* `sit_and_reach_degreeList`
* `standing_long_jump_degreeList`
* `sit_up_degreeList`

---

## 3.3 新增 curl 納入 `isAddFitnessScore`

比照原本 4 項的達標模式，加入 `curl_degreeList` 的達標判斷。

請用與原本相同的 `passStringList` 或既有通過標準，不要自創新規則。

範例方向如下：

```csharp
bool curlPass = false;
foreach (string name in passStringList)
{
    if (curl_degreeList.Contains(name))
    {
        curlPass = true;
        break;
    }
}

if (curlPass)
    addCount++;
```

⚠️ 注意：

* 要與原本其他 4 項的寫法風格一致
* 若原本已有布林變數寫法，就沿用既有風格
* 不要因為新增這段而影響原本 `score += 3` 的邏輯

---

## 3.4 新增 pacer 納入 `isAddFitnessScore`

同樣比照原本邏輯，加入 `pacer_degreeList` 的達標判斷：

```csharp

foreach (string name in passStringList)
{
    if (pacer_degreeList.Contains(name))
    {
        pacerPass = true;
        break;
    }
}

    addCount++;
```

---

## 3.5 保持門檻仍為 4 項

`isAddFitnessScore` 的門檻仍維持：

```csharp

    isAddFitnessScore = true;
```

這次改的是：

* **從原本 4 項選 4**
* 變成 **6 項選 4**

不是改成 5 項或 6 項才加分。
